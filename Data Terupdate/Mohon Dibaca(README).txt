[Indonesia]
Cara penggunaan aplikasi:

!! Pastikan sudah menginstall php di komputer sebelum melanjutkan ke langkah selanjutnya

XAMPP:
1) Copy paste TelkomIndonesia.jpg, Mohon Dibaca(README).txt, HomepageDataTerupdate.php, ScriptDataTerupdate, 
   dan IconWebsite ke folder 'htdocs' di xampp (Bukan folder Data Terupdate nya)
2) Ketik "localhost/HomepageDataTerupdate.php" di web browser

Command Prompt (cmd) :
1) Buka folder Data Terupdate
2) Klik kiri address directory file 
   di sebelah kiri kolom search di window folder Data Terupdate
3) Ketik 'cmd' lalu tekan Enter
4) Ketik 'php -S localhost:(nomor bebas sebagai port, misalnya 1963)
5) Ketik 'localhost:(nomor port yang tadi diketik)/HomepageDataTerupdate.php' di web browser, 
   jika nomor port sesuai contoh berarti ketik 'localhost:1963/HomepageDataTerupdate.php'

Penggunaan di Homepage : 
   1) Kolom Host, Port, Database, dan User jika diisi dengan benar, akan muncul link untuk data terupdate
      Default Host nya adalah "pen-dev.udata.id" dengan Database "web", Port "14113", dan User "u_web"

   2) Tahun dapat diganti dengan cara mengklik panah atas (untuk menambahkan) 
   dan bawah (untuk mengurangi) disebelah kanan tahun atau dihapus dan diganti tahunnya. 

   3) Pastikan nomor disebelah kanan teks 'Tahun' dan 
     'TANGGAL, TOTAL ROW PER BULAN DAN TOTAL ROW KESELURUHAN TAHUN ' sama

   Tombol nomor halaman dapat di mouse hover 
   untuk mengetahui nama-nama BUMN yang akan ditampilkan di halaman tersebut
 
[English]
How to use the application :

!! Make sure the computer have php installed before proceeding to the next step

XAMPP:
1) Copy paste TelkomIndonesia.jpg, Mohon Dibaca(README).txt, HomepageDataTerupdate.php, ScriptDataTerupdate, 
   and IconWebsite into folder 'htdocs' at xampp (Not the folder 'Data Terupdate')
2) Type 'localhost/HomepageDataTerupdate.php' at the web browser

Command Prompt (cmd) :
1) Open folder Data Terupdate' 
2) Left click directory file address 
   at the left of the search box in Data Terupdate folder's window
3) Type 'cmd' then press Enter
4) Type 'php -S localhost:(any number as the port, example 1963)'
5) Type 'localhost:(previously typed port number)/HomepageDataTerupdate.php',
   if the port number matches the example, 
   then type 'localhost:1963/HomepageDataTerupdate.php'

Using the homepage:
   1) Host, Port, Database, and User column if inputted correctly, the link to the most up to date data will be shown. 
      The default Host is pen-dev.udata.id with Database "web", Port "14113", and User "u_web"  

   2) The year can be changed by clicking the up (to increase)
      and down (to decrease) year on the right side of the text 'Tahun' 
      or could be done by erasing and changing the year.

   3) Make sure the number on the right side of 'Tahun' and 
     'TANGGAL, TOTAL ROW PER BULAN DAN TOTAL ROW KESELURUHAN TAHUN ' is the same 

      Page button can be mouse hovered to see the BUMN's name on that page
